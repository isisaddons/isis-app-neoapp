This app demonstrates how Apache Isis can be configured to persist using Neo4J, courtesy of its integration with DataNucleus.

The app itself consists of two domain classes, Host and IpAddress. It also embeds a Neo4J server, accessible at http://localhost:7474..com/apache/isis/blob/master/example/application/simple%61pp/dom/src/main/java/dom/simple/SimpleObject.java" target="_blank">SimpleObject,
For more details, see the Apache Isis website.

To access the app:

  • wicket/

    provides accesses to a generic UI for end-users, Isis' Wicket Viewer. As its name suggests, this viewer is built on top of Apache Wicket™.

  • restful/

    provides access to a RESTful API conformant with the Restful Objects spec. This is part of Isis Core. The implementation technology is JBoss RestEasy.

  • cy2neo/

    provides a lightweight graph DB visualizer.

The default user/password is sven/pass (as configured in the shiro.ini file).
